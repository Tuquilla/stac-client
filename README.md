# Stac-Client

:construction: WIP

Basic implementation of a local running stac browser.

## Run
```bash
go run ./cmd
```

## Build
### Windows
```bash
go build -ldflags "-H windowsgui" -o <name.exe> ./cmd
```

## Requirements

A C compiler is necessary to run the code (e.g. `gcc`).

See also https://docs.fyne.io/started/quick/
